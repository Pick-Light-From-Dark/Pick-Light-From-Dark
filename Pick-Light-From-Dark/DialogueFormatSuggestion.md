# 对话文本格式规范（当前在用）

> 本文档面向**编剧/策划**，说明游戏当前实际使用的 `.txt` 剧本格式规范。文件放在 `Assets/Resources/Dialogue/` 下，编码为 **UTF-8 无 BOM**。

---

## 一、对话

```
角色名：对话内容
```

- 使用**全角冒号** `：`
- 角色名和冒号之间**不要加空格**

```
陆萤：怎么可能？这是我的分数？
宋明月：你还好吗？
```

---

## 二、旁白与场景描述

```
[旁白]：旁白内容
[场景]：场景描述内容
```

两者在游戏中的表现类似，说话人都会显示为"陆萤"。

```
[旁白]：明天是难得的假期，寝室里总是忙碌学习的舍友们终于放松。
[场景]：（回忆白天，八校联考的成绩条被班主任递到手里……）
```

---

## 三、背景指令

### 单层背景
```
[bg:背景图名称]
[bg:背景图名称,过渡效果]
```

过渡效果（可选）：
- `fade` — 淡入淡出
- `slide` — 滑入

```
[bg:the spread quilt]
[bg:quilt aside,slide]
[bg:]                    ← 空值表示清空该层
```

### 多层背景（同时切换）
```
[layer:bg=后景名,mg=中景名,fg=前景名,过渡效果]
```

```
[layer:bg=night_room,mg=,fg=,fade]
```

### 纯色背景（黑屏/白屏）
```
[solid:颜色名]
[solid:颜色名,过渡效果]
```

支持的颜色名：
- `black` / `white` / `red` / `green` / `blue`
- `yellow` / `cyan` / `magenta` / `gray` / `grey`
- `clear` / `transparent`
- 十六进制：`#000000`

```
[solid:black]
[solid:black,fade]
```

> **注意**：背景图名称必须与 `Art` 目录下的实际文件名一致（不含 `.png` 后缀）。编剧写的中文名如果找不到对应文件，需要修改为正确的文件名。

---

## 四、音效与音乐

### 音效
```
[se:音效名]
```

```
[se:DXH_SOUND/04.移动被子]
```

### 背景音乐
```
[bgm:音乐名]
```

```
[bgm:bgm_night]
```

> 音效/BGM 需在 `FungusVNController` 的 Inspector 中提前配置映射，否则运行时左上角会显示红色缺失提示。

---

## 五、选项分支

### 定义选项
```
选项 -选项1 or -选项2。
```

```
选项 -吃 or -不吃。
```

### 选项结果
紧跟选项下方，用 `【选项-吃1】` / `【选项-吃2】` 标记对应结果：

```
【选项-吃1】
陆萤：（因为薯片，我的心情确实变好了一点。）
[新手关卡结束]

【选项-吃2】
陆萤：（那包薯片被我悄悄塞进了宋明月的课桌……）
→【结局一：太阳照常升起】
```

---

## 六、注释

以 `#` 开头的行会被系统忽略，可用于编剧备注：

```
# 这里是注释，不会显示在游戏中
# TODO: 需要补充音效

陆萤：这是实际显示的内容。
```

---

## 七、特殊标记

系统识别但不作为对话显示的标记行：

| 标记 | 说明 |
|---|---|
| `[进入选择界面]` | 提示即将进入选项分支 |
| `[游玩流程结束]` | 当前游玩段落结束 |
| `[结尾剧情]` | 结尾剧情开始 |
| `[新手关卡结束]` | 新手引导结束 |
| `[第二关结束]` / `[第三关结束]` / `[第四关结束]` | 关卡结束标记 |

---

## 八、完整示例

```
# 第一关：宿舍熄灯前
[bg:the spread quilt]

陆萤：（还有十五分钟熄灯。）
陆萤：（……还没有再整理一遍试卷的错题。）

[场景]：（回忆白天，八校联考的成绩条被班主任递到手里……）

陆萤：怎么可能？这是我的分数？

[bg:quilt aside,slide]
[bg:Song_talk_1]
宋明月：你还好吗？

[se:DXH_SOUND/04.移动被子]

# 黑屏闭眼
[solid:black]
[场景]：（表示闭眼）
[场景]：（手电筒的光透过玻璃窗照到陆萤脸上。）

# 选项分支
[进入选择界面]
所有人都睡着了，不会被发现的……要不要试着吃掉它？

选项 -吃 or -不吃。

【选项-吃1】
陆萤：（因为薯片，我的心情确实变好了一点。）
[新手关卡结束]

【选项-吃2】
陆萤：（那包薯片被我悄悄塞进了宋明月的课桌……）
→【结局一：太阳照常升起】
```

## 九、注意事项

1. **空行**：空行会被自动忽略。
2. **空格**：指令内不要加空格，如 `[bg: the spread quilt]` 会解析失败。正确：`[bg:the spread quilt]`。
3. **全角冒号**：对话行必须使用全角 `：`。
4. **文件编码**：务必保存为 **UTF-8 无 BOM**。
5. **素材缺失**：背景图、音效、BGM 名称若未在游戏中配置，运行时左上角会显示红色缺失提示。
6. **黑屏检测**：文本内容中出现"黑屏"二字时，系统会自动将所有背景层切换为纯黑。也可显式使用 `[solid:black]` 指令。
