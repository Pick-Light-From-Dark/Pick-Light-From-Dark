# 视觉小说剧本书写指南

> 本文档面向**编剧/美工**，说明当前游戏对话系统支持的 txt 格式与指令规范。
> 剧本文件放在 `Assets/Resources/Dialogue/` 目录下，`.txt` 后缀，**UTF-8 无 BOM** 编码。

---

## 一、基础格式

### 1. 普通对话
```
角色名：对话内容
```
示例：
```
陆萤：怎么可能？这是我的分数？
宋明月：你还好吗？
```

### 2. 旁白
```
[旁白]：旁白内容
```
示例：
```
[旁白]：明天是难得的假期，寝室里总是忙碌学习的舍友们终于放松。
```

### 3. 场景描述
```
[场景]：场景描述内容
```
示例：
```
[场景]：（回忆白天，八校联考的成绩条被班主任递到手里……）
```

> **旁白与场景的区别**：两者在表现上类似，系统会将说话人显示为"陆萤"，内容按场景描述处理。

## 二、背景指令

### 1. 切换单层背景
```
[bg:背景图名称]
[bg:背景图名称,过渡效果]
```

支持的过渡效果：
- `fade` — 淡入淡出
- `slide` — 滑入

示例：
```
[bg:the spread quilt]
[bg:quilt aside,slide]
[bg:]                    ← 空值，表示清空该层
```

### 2. 同时切换多层背景（高级）
```
[layer:bg=后景名,mg=中景名,fg=前景名,过渡效果]
```

示例：
```
[layer:bg=night_room,mg=,fg=,fade]
```
> 未指定的层保持原样；显式空字符串表示清空该层。

### 3. 纯色背景（黑屏/白屏等）
```
[solid:颜色名]
[solid:颜色名,过渡效果]
```

支持的颜色名：
- `black` / `white` / `red` / `green` / `blue`
- `yellow` / `cyan` / `magenta` / `gray` / `grey`
- `clear` / `transparent`
- `#RRGGBB` 十六进制色值，如 `#000000`

示例：
```
[solid:black]
[solid:black,fade]
```

---

## 三、音效与音乐

### 1. 播放音效
```
[se:音效名]
```
示例：
```
[se:DXH_SOUND/04.移动被子]
```

### 2. 播放背景音乐
```
[bgm:音乐名]
```
示例：
```
[bgm:bgm_night]
```

> 音效/BGM 需在 `FungusVNController` 的 Inspector 中配置映射，否则运行时左上角会显示"素材未找到"提示。

---

## 四、选项分支

### 1. 定义选项
```
选项 -选项1 or -选项2。
```
示例：
```
选项 -吃 or -不吃。
```

### 2. 选项结果
选项下方紧跟结果标记，系统会自动把结果文本绑定到对应选项：
```
【选项-吃1】
陆萤：（因为薯片，我的心情确实变好了一点。）
[新手关卡结束]

【选项-吃2】
陆萤：（那包薯片被我悄悄塞进了宋明月的课桌……）
→【结局一：太阳照常升起】
→【结局描述：薯片改变不了任何事，你也是。】
```

---

## 五、特殊标记

| 标记 | 说明 |
|---|---|
| `[进入选择界面]` | 提示即将进入选项分支 |
| `[游玩流程结束]` | 当前游玩段落结束，后续为剧情 |
| `[结尾剧情]` | 结尾剧情开始 |
| `[新手关卡结束]` | 新手引导结束 |
| `[第二关结束]` / `[第三关结束]` / `[第四关结束]` | 关卡结束标记 |

## 六、注释（新增）

以 `#` 开头的行会被系统忽略，可用于编剧备注：
```
# 这里是注释，不会显示在游戏中
# TODO: 需要补充音效

陆萤：这是实际显示的内容。
```

## 七、完整示例

```
# 第一关：宿舍熄灯前
[bg:the spread quilt]

陆萤：（还有十五分钟熄灯。）
陆萤：（……还没有再整理一遍试卷的错题。）

[场景]：（回忆白天，八校联考的成绩条被班主任递到手里……）

陆萤：怎么可能？这是我的分数？

[bg:quilt aside,slide]
[bg:Song_talk_1]
宋明月：你还好吗？

[se:DXH_SOUND/04.移动被子]

[场景]：（黑屏，表示闭眼）
[solid:black]

[场景]：（手电筒的光透过玻璃窗照到陆萤脸上。）

# 选项分支
[进入选择界面]
所有人都睡着了，不会被发现的……要不要试着吃掉它？

选项 -吃 or -不吃。

【选项-吃1】
陆萤：（因为薯片，我的心情确实变好了一点。）
[新手关卡结束]

【选项-吃2】
陆萤：（那包薯片被我悄悄塞进了宋明月的课桌……）
→【结局一：太阳照常升起】
```

## 八、注意事项

1. **空行**：空行会被自动忽略，不影响解析。
2. **空格**：指令内**不要**随意加空格，如 `[bg: the spread quilt]`（冒号后有空格）可能导致解析失败。正确写法：`[bg:the spread quilt]`。
3. **全角冒号**：对话行必须使用全角冒号 `：`（中文输入法下的冒号），不要用半角 `:`。
4. **文件编码**：务必保存为 **UTF-8 无 BOM**，否则中文可能乱码。
5. **素材缺失**：背景图、音效、BGM 名称若在游戏中未配置，运行时左上角会显示红色"素材未找到"提示。
6. **背景图尺寸**：对话底图（如 `dialog_dark.png`）建议与 CG 图同尺寸（如 1920x1080），上半部分渐变透明，下半部分实色，以适配底部对话框区域。
